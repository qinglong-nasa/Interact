Our app aims to interact with 3D-models on PC using gestures and voice commands. 
       
                  The user will be able to load a 3D-model (like a 3D-model for astronaut,moon ,... etc) and pan, rotate and zoom in/out it on PC.
     
                  we will use a smart phone as an interface between the user and PC.
